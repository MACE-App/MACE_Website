# Installation

Getting started with MACE is straightforward. Follow these steps to install and launch the application.

## Download

1. Visit the [MACE Releases page](https://github.com/mace-app/mace/releases)
2. Download the latest `.dmg` file
3. Open the downloaded file

## Install

1. Drag the MACE app to your Applications folder
2. Eject the disk image

## First Launch

1. Open MACE from your Applications folder
2. If you see a security warning, go to **System Settings > Privacy & Security** and click "Open Anyway"
3. MACE will launch and you can start creating your first project

## Updating MACE

Currently, updates are manual:

1. Download the latest version from GitHub
2. Replace the existing app in your Applications folder

:::note
Automatic updates are planned for a future release.
:::

## Uninstalling

To remove MACE:

1. Quit the application if running
2. Drag MACE from Applications to Trash
3. Optionally, remove project files (`.mace` files) from your documents

## Troubleshooting

### App Won't Open

If macOS prevents the app from opening:

1. Go to **System Settings > Privacy & Security**
2. Scroll to the Security section
3. Click "Open Anyway" next to the MACE message

### Missing Features

Some features may be disabled in alpha releases. Check the [GitHub Issues](https://github.com/mace-app/mace/issues) for known limitations.
