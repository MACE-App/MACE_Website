## Available Formats

### PDF Reports

Professional, print-ready compliance reports featuring:

- **Cover page** with baseline and device info
- **Executive summary** with pass/fail statistics
- **Charts** showing compliance breakdown
- **Detailed findings** for each rule
- **Section summaries** with subtotals
- **Page numbers** and headers

### HTML Reports

Interactive, web-based reports with:

- **Responsive design** - Works on any device
- **Interactive filtering** - Toggle visibility by status
- **Search** - Find specific rules
- **Charts** - Visual compliance breakdown
- **Expandable details** - Click to see more
- **Print-friendly** - Clean output when printed

### CSV Export

Spreadsheet-friendly format including:

- **One row per rule** - Easy filtering and sorting
- **All fields** - Rule ID, title, status, expected, actual
- **Summary header** - Totals and percentages
- **Device info** - Serial, model, hostname
- **Comments** - User annotations preserved

| Column | Content |
|--------|---------|
| Rule ID | Unique identifier |
| Title | Rule name |
| Section | Category |
| Status | Pass/Fail/etc. |
| Expected | Expected value |
| Actual | Found value |
| STIG ID | DISA reference |
| CIS ID | CIS reference |
| Comments | User notes |
| Override | If manually changed |

### DISA STIG CKL

Checklist format for STIG Viewer compatibility:

- **Automatic STIG ID mapping** - Links findings to STIG IDs
- **Proper CKL XML structure** - Valid checklist format
- **Status translation:**
  - Pass = NotAFinding
  - Fail = Open
  - Error = Open
  - Manual = Not_Reviewed
  - N/A = Not_Applicable
- **Comments preserved** - Notes included in finding details

## Exporting Results

1. Complete an audit
2. Click **Export** button
3. Select format(s) to generate
4. Configure options
5. Choose save location
6. Click **Export**

Multiple formats can be generated simultaneously.

## Report Contents

### Included Information

Each report contains:

| Section | Content |
|---------|---------|
| **Header** | Title, date, author, benchmark |
| **Device Info** | Serial, model, OS, hostname |
| **Summary** | Pass/fail counts and percentages |
| **By Section** | Grouped results with subtotals |
| **Details** | Per-rule findings and values |
| **Overrides** | Manually changed statuses |
| **Comments** | User annotations |

### Charts (PDF/HTML)

- **Pie chart** - Overall pass/fail distribution
- **Bar chart** - Status breakdown by section
- **Progress indicators** - Compliance percentage

## Configuration Options

### Display Options

| Option | Description |
|--------|-------------|
| **Show Rule ID** | Display MACE rule identifiers |
| **Show STIG ID** | Display DISA STIG references |
| **Show CIS ID** | Display CIS benchmark references |
| **Include Passed** | Include passing rules in report |
| **Include Sections** | Group by section with summaries |
| **Include Charts** | Add visual statistics |

### Branding Options (MACE Engine)

| Option | Description |
|--------|-------------|
| **Custom Logo** | Your organization's logo |

| **Show MACE Branding** | Include "Generated by MACE" |
| **Footer Text** | Custom footer message |
